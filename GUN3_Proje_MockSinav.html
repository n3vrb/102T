1 Dictionary.java — Final Projesi Şablonu

Aşağıdaki şablonu sıfırdan, şablona bakmadan yaz. Sonra karşılaştır.

import java.util.*;

// ========= KeyNotFoundException =========
class KeyNotFoundException extends RuntimeException {
    private Object key;

    public KeyNotFoundException(Object key) {
        this.key = key;
    }

    @Override
    public String toString() {
        return "KeyNotFoundException Key = " + key;
    }
}

// ========= Dictionary (Version 1 — HashMap ile) =========
public class Dictionary<K, V> {
    private Map<K, V> map;

    public Dictionary() {
        map = new LinkedHashMap<K, V>(); // LinkedHashMap sırayı korur
    }

    public V get(K key) {
        if (!map.containsKey(key)) throw new KeyNotFoundException(key);
        return map.get(key);
    }

    public void update(K key, V value) {
        map.put(key, value); // put: varsa günceller, yoksa ekler
    }

    public Dictionary<K, V> copy() {
        Dictionary<K, V> newDict = new Dictionary<K, V>();
        newDict.map.putAll(this.map);
        return newDict;
    }

    public Collection<K> keys() {
        return map.keySet();
    }

    public void clear() {
        map.clear();
    }

    @Override
    public String toString() {
        return map.toString(); // Otomatik {K=V, K=V} formatı verir!
    }
}
💡 Version 2 (ArrayList tabanlı): Hoca iki versiyon istiyor. Version 2'de HashMap/TreeMap kullanma. Bunun yerine iki ArrayList tut: ArrayList<K> keys ve ArrayList<V> values. Aynı index'teki key-value eşli. get() için linear search yap.

Sıfırdan Yazma Kontrol Listesi

  • KeyNotFoundException extends RuntimeException yazıldı mı?
  • KeyNotFoundException constructor'ı Object key parametre alıyor mu?
  • KeyNotFoundException toString() "KeyNotFoundException Key = " + key formatında mı?
  • Dictionary class'ı <K, V> generic parametreli mi?
  • get() → key yoksa exception fırlatıyor mu?
  • update() → hem ekleme hem güncelleme için aynı metod mu?
  • copy() → derin kopya mı yapıyor?
  • keys() → Collection döndürüyor mu?
  • toString() → {K=V, K=V} formatı mı?

2 University.java — Bütünleme Projesi Şablonu

import java.util.*;

// ========= StudentNotFoundException =========
class StudentNotFoundException extends RuntimeException {
    private Object student;
    public StudentNotFoundException(Object s) { student = s; }
    @Override
    public String toString() {
        return "StudentNotFoundException - Student = " + student;
    }
}

// ========= CourseNotFoundException =========
class CourseNotFoundException extends RuntimeException {
    private Object course;
    public CourseNotFoundException(Object c) { course = c; }
    @Override
    public String toString() {
        return "CourseNotFoundException - Course = " + course;
    }
}

// ========= University<S, C, G> =========
public class University<S, C, G> {
    private String name;
    private Map<S, Map<C, G>> data;  // öğrenci → {ders → not}

    public University(String name) {
        this.name = name;
        data = new LinkedHashMap<S, Map<C, G>>();
    }

    public University() {
        this("Unnamed University"); // default isim
    }

    public String getName() { return name; }

    public void register(S student, C course) {
        data.putIfAbsent(student, new LinkedHashMap<C, G>());
        data.get(student).put(course, null); // not henüz verilmedi
    }

    public void grade(S student, C course, G grade) {
        if (!data.containsKey(student))
            throw new StudentNotFoundException(student);
        if (!data.get(student).containsKey(course))
            throw new CourseNotFoundException(course);
        data.get(student).put(course, grade);
    }

    public Collection<S> students(C course) {
        List<S> result = new ArrayList<S>();
        for (Map.Entry<S, Map<C, G>> e : data.entrySet()) {
            if (e.getValue().containsKey(course)) result.add(e.getKey());
        }
        return result;
    }

    public String transcript(S student) {
        StringBuilder sb = new StringBuilder();
        sb.append("Transcript for Student " + student + "\n");
        for (Map.Entry<C, G> e : data.get(student).entrySet()) {
            String g = (e.getValue() == null) ? "N/A" : e.getValue().toString();
            sb.append(e.getKey() + " - " + g + "\n");
        }
        return sb.toString().trim();
    }

    @Override
    public String toString() {
        StringBuilder sb = new StringBuilder();
        sb.append(name + " information\n");
        for (S s : data.keySet()) sb.append(transcript(s) + "\n");
        return sb.toString().trim();
    }
}
⚠️ Dikkat: Bütünleme projesinde üç generic type parametre var: S (student), C (course), G (grade). Hem University<Long, String, Integer> hem de University<String, Integer, String> ile çalışabilmeli!

True / False

Soru 1
ArrayList<double> list = new ArrayList<double>(); — This will have a compile error.
→ Cevabı Göster
✅ TRUE — double primitive type, generic'te kullanılamaz. Double kullanılmalı.
Soru 2
A Stack is a type of First In First Out data structure.
→ Cevabı Göster
❌ FALSE — Stack LIFO'dur (Last In First Out). FIFO olan Queue'dur.
Soru 3
An interface can have concrete (non-abstract) methods.
→ Cevabı Göster
⚠️ Dikkatli ol — Java 8+ ile default methodlar eklenebilir. Ama klasik anlamda sadece abstract + constants → FALSE. Bu derste büyük ihtimalle FALSE kabul edilir.
Soru 4
The toString() method in the Object class is called automatically when an object is passed to println().
→ Cevabı Göster
✅ TRUE — println() objeden otomatik toString() çağırır.
Soru 5
A class with no abstract methods can still be declared as abstract.
→ Cevabı Göster
✅ TRUE — abstract class'ın abstract method zorunluluğu yoktur. Soyut sınıf tanımlamak için yeterli.

Multiple Choice — Hepsini İşaretle

Soru 6
Which of the following are true about a HashMap? (Choose all correct)
  • a. Keys must be unique
  • b. Values must be unique
  • c. Maintains insertion order
  • d. Allows null keys
  • e. get() and put() are O(1) on average
→ Cevabı Göster

  • b. ArrayIndexOutOfBoundsException
  • c. IOException

4 🎯 Mock Sınav — Bölüm 2: Hata Bulma

Compiler Error Analizi

Soru 8 — Hata neden var? Kısaca açıkla.
class Vehicle {
    String brand;
    public Vehicle(String brand) {
        this.brand = brand;
    }
}

class Car extends Vehicle {
    int doors;
    public Car(int doors) {
        this.doors = doors;
    }
}
→ Cevabı Göster
Hata: Vehicle sınıfının sadece String parametreli constructor'ı var, no-arg constructor yok. Car extends Vehicle yapıyor ve Car'ın constructor'ı otomatik olarak super() çağırıyor ama bu constructor Vehicle'da mevcut değil.

Çözüm: Car'ın constructor'ına super("Unknown"); ekle veya Vehicle'a no-arg constructor ekle.
Soru 9 — Düzeltilmiş Car constructor'ını yaz
→ Cevabı Göster
Çözüm 1:
public Car(int doors) { super("Unknown"); this.doors = doors; }

Çözüm 2: Vehicle'a no-arg constructor ekle:
public Vehicle() { this.brand = "Unknown"; }

5 🎯 Mock Sınav — Bölüm 3: Fill in the Blanks (En Ağır Bölüm)

⏱️ Bu bölüm için 30 dakika. Her soruyu elle çöz, sonra açıkla.

Senaryo — Aynı Hoca Tarzı

Aşağıdaki kodu 3 farklı öğrenciye verdim. Boşlukları doldurmalarını istedim. Sadece String class veya kendi yazdıkları MyClass adında bir class kullanabilirler.

public static void main(String[] args) {
    _____________ refVar1 = _____________;
    _____________ refVar2 = _____________;
    System.out.println(refVar1 == refVar2);
    System.out.println(refVar1.equals(refVar2));
}
Öğrenci 1 — Output:
false
true
Boşlukları doldur:
→ Cevabı Göster
String refVar1 = new String("hello");
String refVar2 = new String("hello");
(Farklı objeler, aynı içerik → == false, equals true)
Öğrenci 2 — Output:
true
true
Boşlukları doldur:
→ Cevabı Göster
String refVar1 = new String("hello");
String refVar2 = refVar1;
(Aynı referans → == true, equals true)
Öğrenci 3 — Output:
true
false
Boşlukları doldur. Bu öğrenci kesinlikle kendi class'ını yazmak zorunda mıydı?
→ Cevabı Göster
EVET — kesinlikle custom class yazması gerekiyor!
String ile true/false mümkün değil (== true → aynı referans → equals da true olur).

MyClass refVar1 = new MyClass();
MyClass refVar2 = refVar1;

MyClass şöyle yazılabilir:
class MyClass {
  public boolean equals(Object o) { return false; }
}
Soru — Öğrenci 3'ün kullandığı MyClass implementasyonunu yaz
→ Cevabı Göster
class MyClass {
  @Override
  public boolean equals(Object obj) {
    return false;
  }
}
