🎯 Her Sınavda Kesin Çıkan Konular (Her İkisinde de Göründü)

🔴 #1 KRİTİK — == vs .equals()

Hem final hem bütünlemede ağır puanlı. 4 kombinasyonu ezberle:
false/true → farklı obje, aynı içerik
false/false → farklı obje, farklı içerik
true/true → aynı referans
true/false → SADECE custom class ile mümkün! (equals() override)

🟡 #2 — Abstract Class Tuzakları

Her sınavda yanlış yapılan soru:
✅ Can have abstract methods
✅ Can have non-abstract (concrete) methods
✅ Cannot be instantiated
❌ "Must have abstract methods" → YANLIŞ
❌ "Cannot have concrete methods" → YANLIŞ

🟢 #3 — Generics

ArrayList<int> → COMPILE ERROR (primitive yasak)
ArrayList<Integer> → doğru
Proje ödevi her seferinde generic class tasarımı istiyor.
class MyClass<K, V> { ... }

🔵 #4 — Constructor + Inheritance

Parent'ta no-arg constructor yoksa child'da super(args) zorunlu.
Constructor kuralları:
✅ Default sağlanır, eğer hiç yazılmamışsa
✅ Sınıfla aynı isim
❌ "Tam olarak bir constructor" → YANLIŞ (overload mümkün)
