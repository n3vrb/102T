## 🧠 Key Concepts Quick Reference

### Object-Oriented Programming
- **Encapsulation** — hiding internal state via private fields + getters/setters
- **Inheritance** — `extends` keyword, `super()` calls, method overriding
- **Polymorphism** — dynamic dispatch, `@Override`, upcasting/downcasting
- **Abstraction** — `abstract` classes, `interface` definitions

### Generics & Collections
- **Generics** — `<T>`, bounded wildcards `<? extends T>`, type safety
- **List / Stack / Queue** — `ArrayList`, `LinkedList`, `ArrayDeque`
- **Set / Map** — `HashSet`, `TreeSet`, `HashMap`, `TreeMap`

### Algorithms
- **Sorting** — Bubble, Selection, Insertion, Merge, Quick
- **Complexity** — Big-O notation: O(1), O(log n), O(n), O(n log n), O(n²)
